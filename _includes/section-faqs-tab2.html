How is the contract process conducted?

  • First, a mutual NDA must be signed before finalizing HaloSync contract.
  • Once the contract is finalized, integration with your existing systems will commence. Halo&Co will provide technical support along with an PolarHub API integration development guide to ensure integration is completed as quickly as possible. For detailed integration steps, please refer to Solutions > How HaloSync works.
  • If you prefer to use NDC content without system integration, you can start using NDC content immediately through our Luna(B2B Portal Service). If you're specifically interested in Luna, please contact us separately.

In what format is the PolarHub provided?

After the contract is finalized, integration work with your existing system will proceed. To ensure a swift integration and enable you to start your business as quickly as possible, Halo&Co provides technical support along with an PolarHub API integration development guide.

What do we need to prepare for PolarHub integration?

  • When signing the contract, the travel agency's IATA# and endpoints for both the sandbox and production environments for API integration are required. If you apply through Solutions > Ready to board the future of air travel” we will provide you with detailed integration steps and all necessary information.
  • If you do not have an IATA#, please contact us separately.

How long does it take to integrate with our platform and PolarHub?

• Integration typically takes 2 months, but may vary based on client needs and situation.
• For back-office system integation, expect around 3 months.

How complex is the process of integrating the PolarHub into an existing system?

PolarHub API is designed based on the NDC standard to enable easy integration with OTA (Online Travel Agency) systems. Through a single API connection, you can quickly and efficiently access NDC content from multiple airlines, reducing both development time and cost.
